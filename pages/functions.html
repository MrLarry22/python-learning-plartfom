🧠 Python Functions: A Complete Guide for Beginners

🔍 What is a Function?

A function is a reusable block of code that performs a specific task. Functions help organize programs into smaller, manageable parts.

🛠️ 1. Defining a Function

✅ Syntax:

def function_name():
    # code block

💡 Example:

def greet():
    print("Hello, Python learner!")

📞 Calling the Function:

greet()  # Output: Hello, Python learner!

🧳 2. Parameters and Arguments

✅ Syntax with Parameters:

def greet(name):
    print(f"Hello, {name}!")

greet("Alice")  # Output: Hello, Alice!

🔄 Types of Arguments:

TypeExampleDescription
Positionalgreet("Bob")Matches by position
Keywordgreet(name="Tom")Matches by name
Defaultdef greet(name="Guest")Uses default if not provided
Variable-length*args, **kwargsMultiple positional or keyword args

🧪 Example with *args and **kwargs:

def show_details(*args, **kwargs):
    print(args)
    print(kwargs)

show_details(1, 2, name="Alice", age=25)

🎯 3. Return Statement

📦 Purpose:

Functions can return a value using the return statement.

def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # Output: 8

📌 Note: Functions without return implicitly return None.

🔐 5. Variable Scope

🧠 Scope Rules:

ScopeDescription
LocalDefined inside a function
GlobalDefined outside all functions

⚠️ Example:

x = 10  # Global

def func():
    x = 5  # Local
    print("Inside:", x)

func()
print("Outside:", x)

Output:

Inside: 5
Outside: 10

🔓 Using global:

x = 10

def modify():
    global x
    x = 20

modify()
print(x)  # Output: 20

⚡ 6. Lambda (Anonymous) Functions

Short functions without a name. Used for small, one-time operations.

square = lambda x: x ** 2
print(square(4))  # Output: 16

📚 7. Function Documentation (Docstrings)

To explain what a function does. Helps others (and you) understand your code later.

def multiply(a, b):
    """Multiply two numbers and return the result."""
    return a * b

help(multiply)

🧪 11. Practice Exercises

✅ Greeting Function

def greet_user(name):
    print(f"Hello, {name}!")

greet_user("Leo")

✅ Sum Two Numbers

def add(a, b):
    return a + b

print(add(10, 20))

🧠 Factorial Function

def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n - 1)

print(factorial(5))

🧠 Variable Argument Function

def total(*numbers):
    return sum(numbers)

print(total(1, 2, 3, 4))

🧠 Lambda and Sorting

items = [('apple', 3), ('banana', 2), ('cherry', 5)]
sorted_items = sorted(items, key=lambda item: item[1])
print(sorted_items)
