🧮 Python Variables and Data Types

📌 1. What is a Variable?

A variable is a name that refers to a value stored in memory.

name = "Alice"
age = 25

🏷️ 2. Naming Variables

✅ Valid Names:

  • Begin with a letter or underscore
  • Contain letters, digits, and underscores
  • Are case-sensitive
first_name = "John"
_firstName = "Amy"
user2 = "Kevin"

❌ Invalid Names:

2name = "Wrong"
user-name = "Error"
class = "Keyword"

✅ Best Practices:

  • Use descriptive names
  • Use snake_case

🧠 3. Python is Dynamically Typed

You don't declare data types explicitly—Python infers them.

age = 20
pi = 3.14
name = "Alice"
x = 5
x = "Now I'm a string"

🔢 4. Core Data Types

1️⃣ Integers

score = 100
temperature = -10

2️⃣ Floats

height = 5.9
average = 4.5

3️⃣ Strings

greeting = "Hello"
language = 'Python'
bio = """This is
a multi-line string."""

4️⃣ Booleans

is_admin = True
is_logged_in = False

🧩 5. Complex Data Types

📋 Lists

fruits = ["apple", "banana", "cherry"]
print(fruits[0])

🔐 Tuples

colors = ("red", "green", "blue")

🧰 Dictionaries

person = {"name": "Alice", "age": 25}
print(person["name"])

🔁 Sets

unique_numbers = {1, 2, 3, 3}
print(unique_numbers)

🔎 6. Type Checking and Conversion

x = 5
print(type(x))
str_num = "10"
int_num = int(str_num)
float_num = float("3.14")
string = str(123)
boolean = bool(0)

🌍 7. Variable Scope

🔹 Local

def greet():
    message = "Hello"
    print(message)

🔸 Global

language = "Python"
def show():
    print(language)

⚠️ Modifying Global

count = 0
def increment():
    global count
    count += 1

🧹 8. Memory Management

a = [1, 2, 3]
b = a
del a

🌟 11. Best Practices

  • ✅ Use descriptive names
  • ✅ snake_case convention
  • ❌ Avoid one-letter names unless needed
  • ✅ Initialize variables before use
  • ✅ Use type checking

🧠 Final Advice

Variables are the building blocks of your code. Understand how to name, assign, and use them properly—and you’ll have unlocked the first major milestone in your Python journey.
