🧱 Python Object Oriented Programming (OOP): A Complete Guide

🔍 What is Object Oriented Programming?

Object Oriented Programming (OOP) is a programming paradigm based on the concept of "objects", which can contain data (attributes) and functions (methods).

Python is an object-oriented language, meaning almost everything in Python is an object!

🧩 1. Classes and Objects

🧠 Definition

  • Class: A blueprint for creating objects.
  • Object: An instance of a class.

✅ Syntax

class Dog:
    pass

my_dog = Dog()

🐶 Real-World Analogy

Think of a class as the blueprint for a house, and an object as the actual house built from that blueprint.

🧰 2. Attributes and Methods

🧠 Attributes

class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

⚙️ Methods

    def bark(self):
        print(f"{self.name} says Woof!")
dog1 = Dog("Buddy", 3)
dog1.bark()  # Output: Buddy says Woof!

🏗️ 3. Constructor Method (__init__)

🧠 Purpose

The __init__() method runs automatically when an object is created.

class Car:
    def __init__(self, model):
        self.model = model

car1 = Car("Toyota")
print(car1.model)  # Output: Toyota

🔐 4. Encapsulation & Data Hiding

🧠 What is Encapsulation?

Encapsulation is the bundling of data and methods into a single unit — the object.

🔒 Private Attributes

class BankAccount:
    def __init__(self, balance):
        self.__balance = balance  # private attribute

    def get_balance(self):
        return self.__balance

🚫 Pitfall

__balance is name-mangled and becomes _ClassName__balance.

🧬 5. Inheritance

🧠 Definition

class Animal:
    def speak(self):
        print("Animal speaks")

class Dog(Animal):
    def speak(self):
        print("Dog barks")

pet = Dog()
pet.speak()  # Output: Dog barks

🔁 Types of Inheritance

  • Single
  • Multiple
  • Multilevel
  • Hierarchical

🧠 Multiple Inheritance Example

class A:
    def show(self):
        print("A")

class B:
    def display(self):
        print("B")

class C(A, B):
    pass

obj = C()
obj.show()
obj.display()

🎭 6. Polymorphism

🧠 What is it?

class Cat:
    def sound(self):
        print("Meow")

class Dog:
    def sound(self):
        print("Bark")

for pet in (Cat(), Dog()):
    pet.sound()

🧼 7. Abstract Classes and Methods

from abc import ABC, abstractmethod

class Vehicle(ABC):
    @abstractmethod
    def start(self):
        pass

class Car(Vehicle):
    def start(self):
        print("Car engine started")

🧙‍♂️ 8. Special (Magic/Dunder) Methods

class Book:
    def __init__(self, title):
        self.title = title

    def __str__(self):
        return f"Book: {self.title}"

b = Book("Python 101")
print(b)  # Output: Book: Python 101

🧱 9. Class Relationships: Composition vs Aggregation

🧠 Composition

class Engine:
    def start(self):
        print("Engine starting...")

class Car:
    def __init__(self):
        self.engine = Engine()

🧠 Aggregation

class Engine:
    pass

class Car:
    def __init__(self, engine):
        self.engine = engine

📏 10. Best Practices

✅ Practice💬 Why It Matters
Use __init__ for initializationEnsures objects start with valid state
Keep methods short and focusedImproves readability and maintainability
Use encapsulation for data safetyAvoids accidental data corruption
Favor composition over inheritanceIncreases flexibility and reusability
Use meaningful namesMakes code self-explanatory

🧪 Practice Exercises

✅ Beginner: Create a Class

class Student:
    def __init__(self, name):
        self.name = name

    def greet(self):
        print(f"Hello, {self.name}!")

s = Student("Alice")
s.greet()

✅ Intermediate: Inheritance Example

class Person:
    def __init__(self, name):
        self.name = name

class Employee(Person):
    def __init__(self, name, job):
        super().__init__(name)
        self.job = job

e = Employee("John", "Engineer")
print(e.name, e.job)

✅ Advanced: Abstract Class Challenge

from abc import ABC, abstractmethod

class Shape(ABC):
    @abstractmethod
    def area(self):
        pass

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def area(self):
        return 3.14 * self.radius ** 2

c = Circle(3)
print(c.area())
