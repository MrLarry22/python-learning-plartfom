🔰 Introduction to Python Programming

🧠 What is Python?

Python is a high-level, general-purpose programming language known for its simplicity and readability. It was created by Guido van Rossum and first released in 1991.

Whether you're a beginner learning your first language or a professional building complex systems, Python has the tools and libraries to support you.

🛠️ Installing Python

  1. Go to the official Python website
  2. Click Download Python 3.x.x
  3. Run the installer and check "Add Python to PATH"
  4. Verify in terminal: 
    python --version

🔁 Control Structures

🧪 If-Else

age = int(input("Enter your age: "))
if age >= 18:
    print("You're an adult.")
else:
    print("You're a minor.")

🔄 Loops

for i in range(5):
    print("Step", i)

count = 0
while count < 3:
    print("Counting", count)
    count += 1
