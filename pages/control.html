🧭 Python Control Structures: A Complete Guide for Beginners

🔍 What are Control Structures?

Control structures govern the flow of execution in a Python program. Instead of running code line-by-line top to bottom, control structures let you make decisions, repeat actions, or skip parts based on conditions.

📘 1. Conditional Statements

🧠 Concept:

Conditional statements allow your program to make decisions.

✅ Syntax:

if condition:
    # code block
elif another_condition:
    # another block
else:
    # fallback block

🚀 Example:

age = 18
if age >= 18:
    print("You're eligible to vote.")
else:
    print("Sorry, you're too young to vote.")

💡 Real-World Use:

  • Checking login credentials
  • Determining pricing tiers
  • Running feature toggles

⚠️ Common Pitfall:

❌ Forgetting colons (`:`) or incorrect indentation.

🔁 2. Loops

🧵 A. for Loop

for item in iterable:
    # code block

✅ Example:

fruits = ['apple', 'banana', 'cherry']
for fruit in fruits:
    print(f"I love {fruit}!")

🎯 Use Cases:

  • Iterating through files
  • Displaying data from a list
  • Running a function multiple times

🔄 B. while Loop

while condition:
    # code block

✅ Example:

count = 1
while count <= 5:
    print(count)
    count += 1

🧠 Good For:

  • Waiting for user input
  • Retry logic
  • Games or event loops

🧩 3. Loop Control Statements

KeywordFunction
breakStops the loop completely
continueSkips the current iteration
passDoes nothing (placeholder)

🔨 break Example:

for num in range(1, 10):
    if num == 5:
        break
    print(num)

🪁 continue Example:

for num in range(1, 6):
    if num == 3:
        continue
    print(num)

💬 pass Example:

for letter in 'abc':
    pass

🧱 4. Nested Control Structures

for i in range(1, 4):
    if i % 2 == 0:
        print(f"{i} is even")
    else:
        print(f"{i} is odd")

🔂 Nested Loops:

for i in range(1, 3):
    for j in range(1, 3):
        print(f"{i} x {j} = {i*j}")

🧠 Best Practices

✅ Best Practice⚠️ Why It Matters

🧪 Try It Yourself

🔁 Voting System:

age = int(input("Enter your age: "))
if age >= 18:
    print("✅ You can vote.")
else:
    print("❌ You must be at least 18 to vote.")

🔂 Countdown with while:

num = 5
while num > 0:
    print(num)
    num -= 1
print("🚀 Liftoff!")

🔁 Even Numbers:

for i in range(1, 11):
    if i % 2 != 0:
        continue
    print(i)

🧱 Nested Loops Grid:

for row in range(1, 4):
    for col in range(1, 4):
        print(f"({row}, {col})", end=" ")
    print()
